# Iceland-2008
Project looking into macro fundamentals pre-, during and post- the 2008 financial crisis in Iceland, specifically with respect to monetary policy and debt restructuring. Semester 6 project for Advance Macroecon course curriculum.

Models:
1. ADF tests 
2. Granger causality tests
3. VAR + Impulse Response Functions
4. Chow test for crisis period
5. Volatility in periods comparison
6. t means test for GDP growth
7. OLS regression
8. Philips Curve
9. Other visualizations
